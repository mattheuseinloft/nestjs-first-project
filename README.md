# Nest Base Project

This base project has a structure slighty different from the default used by the NestJS CLI. Therefore, CLI generated files should be modified and be placed in the correct location accordingly.

This project uses eslint configured with certain options to enhance code maintainability and to make good use of Typescript type system.

You might consider to fork this project and build your API on top of it.

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
